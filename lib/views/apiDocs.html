The API functionality is namespace under -- /api/v1

The possible Walk Thru endpoints are:

/walk_thrus -- GET -- all Walk Thrus /walk_thrus -- POST -- create a Walk Thru /walk_thrus/:id -- GET -- one Walk Thru

The possible Step endpoints are:

/steps -- POST -- create a Step /steps/:id -- GET -- one Step

The possible Categories endpoints are:

/categories -- GET -- all Categories
